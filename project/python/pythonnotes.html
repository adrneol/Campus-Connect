1. Variables and Data Types

Python supports dynamic typing, meaning you don‚Äôt need to declare the type of a variable. Common data types include integers, floats, strings, booleans, lists, tuples, dictionaries, and sets.

# Declaring variables name = "Alice" # String age = 25 # Integer height = 5.6 # Float is_student = True # Boolean my_list = [1, 2, 3] # List my_tuple = (4, 5) # Tuple my_dict = {"name": "Alice", "age": 25} # Dictionary my_set = {1, 2, 3} # Set # Printing types print(type(name)) # print(type(age)) #

2. Control Structures

Control structures like if-else, for loops, and while loops help manage the flow of your program.

# If-else statement score = 85 if score >= 90: print("Grade: A") elif score >= 80: print("Grade: B") else: print("Grade: C") # For loop for i in range(5): print(f"Number: {i}") # While loop count = 0 while count < 3: print(f"Count: {count}") count += 1

3. Functions

Functions allow you to modularize your code. You can define functions with parameters and return values.

# Defining a function def greet(name): return f"Hello, {name}!" # Function with default parameter def add(a, b=10): return a + b # Calling functions print(greet("Bob")) # Hello, Bob! print(add(5)) # 15 print(add(5, 20)) # 25

4. Lists and List Comprehension

Lists are mutable sequences. List comprehensions provide a concise way to create lists.

# Creating a list numbers = [1, 2, 3, 4, 5] # List operations numbers.append(6) # Add to end numbers.pop() # Remove last element print(numbers) # [1, 2, 3, 4, 5] # List comprehension squares = [x**2 for x in range(5)] print(squares) # [0, 1, 4, 9, 16] # Filter with list comprehension evens = [x for x in range(10) if x % 2 == 0] print(evens) # [0, 2, 4, 6, 8]

5. Dictionaries

Dictionaries store key-value pairs and are useful for mapping data.

# Creating a dictionary student = {"name": "Charlie", "age": 20, "grades": [85, 90, 88]} # Accessing values print(student["name"]) # Charlie # Adding/Updating keys student["major"] = "Computer Science" print(student) # {'name': 'Charlie', 'age': 20, 'grades': [85, 90, 88], 'major': 'Computer Science'} # Looping through dictionary for key, value in student.items(): print(f"{key}: {value}")

6. File Handling

Python makes it easy to read from and write to files using built-in functions.

# Writing to a file with open("example.txt", "w") as file: file.write("Hello, Python!\n") file.write("This is a test.") # Reading from a file with open("example.txt", "r") as file: content = file.read() print(content) # Hello, Python! # This is a test. # Appending to a file with open("example.txt", "a") as file: file.write("\nAppending this line.")

7. Object-Oriented Programming (OOP)

Python supports OOP with classes and objects for structured programming.

# Defining a class class Dog: def __init__(self, name, age): self.name = name self.age = age def bark(self): return f"{self.name} says Woof!" # Creating an object my_dog = Dog("Buddy", 3) print(my_dog.name) # Buddy print(my_dog.bark()) # Buddy says Woof! # Inheritance class Puppy(Dog): def play(self): return f"{self.name} is playing!" my_puppy = Puppy("Max", 1) print(my_puppy.bark()) # Max says Woof! print(my_puppy.play()) # Max is playing!

8. Exception Handling

Exception handling ensures your program doesn‚Äôt crash due to errors.

# Try-except block try: x = 10 / 0 except ZeroDivisionError: print("Cannot divide by zero!") finally: print("This runs no matter what.") # Multiple exceptions try: num = int(input("Enter a number: ")) print(10 / num) except ValueError: print("Please enter a valid number!") except ZeroDivisionError: print("Cannot divide by zero!")

10. Lambda Functions

Lambda functions are small, anonymous functions defined with the lambda keyword.

# Basic lambda function add = lambda x, y: x + y print(add(3, 5)) # 8 # Using lambda with map numbers = [1, 2, 3, 4] squared = list(map(lambda x: x**2, numbers)) print(squared) # [1, 4, 9, 16] # Using lambda with filter evens = list(filter(lambda x: x % 2 == 0, numbers)) print(evens) # [2, 4]
